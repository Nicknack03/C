# C Programming Programs

A collection of C programming programs covering fundamental concepts, data structures, algorithms, matrix operations, number programs, and pattern printing.

This repository is useful for:

- 🎓 College Practical Exams
- 💻 C Programming Practice
- 📚 Interview Preparation
- 🚀 Beginners Learning C

## Requirements

- GCC Compiler
- Visual Studio Code (Recommended)

---

## Compile

```bash
gcc filename.c -o program
```

Example

```bash
gcc factorial.c -o factorial
```

---

## Run

### Windows

```bash
factorial.exe
```

### Linux/macOS

```bash
./factorial
```

---

## Example

Compile

```bash
gcc armstrong_number.c -o armstrong
```

Run

```bash
armstrong.exe
```

Output

```
Enter a number: 153

153 is an Armstrong Number
```
